C++ wrapper for Zend API

## Requirements

- PHP 7.2 or later
- Linux/MacOS/Windows
- GCC 4.8 or later
- Composer

## Build phpx (bin)
```shell
./build.sh
sudo cp bin/phpx /usr/local/bin
```

## Build libphpx.so
```shell
cmake .
make -j 4
sudo make install
sudo ldconfig
```

## Create Project
```shell
phpx create cpp_ext
```

## Generate ArgInfo & Function Entires
```shell
php script/gen_stub.php your_stub_dir
```

## Build extension
```shell
cd examples/cpp_ext
phpx build -v
sudo phpx install
```

## Load your extension
Edit `php.ini`, add `extension=cpp_ext.so`

## Run
```shell
php echo.php
```
